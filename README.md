# pbot

pan93412's extensible user bot, which is full-documented, engineered and based on Actor model.

## Usage

### Build

```sh
cargo run --release
```

### Configure

1. Copy `.env.example` to `.env`
2. Configure it according to the instruction.

## Hacking

### Build

```sh
cargo build
```

### Docs

```sh
cargo doc
```

### Run

```sh
cargo run
```
